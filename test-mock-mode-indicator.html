Implementation Details

Trigger Condition: 
settings.aiEngine === 'mock'
Visual Elements:
  • Orange background (bg-orange-50) with orange border (border-orange-200)
  • Orange text color (text-orange-700)
  • Pulsing orange dot (animate-pulse) for visual attention
  • Underlined "Change in Settings" link
User Actions:
  • Clicking "Change in Settings" opens the settings popup
  • Settings button remains accessible for other configurations
  • Indicator disappears when switching to real AI engines
Location:
  • Positioned right before the settings button
  • Uses margin-right: 8px to maintain spacing
  • Integrates naturally with existing header controls

User Experience Benefits

✅ Clear Communication: Users immediately understand they're in demo mode
✅ Easy to Change: Direct link to settings for switching to real AI
✅ Non-intrusive: Visible but doesn't overwhelm the interface
✅ Discoverable: Located near settings where users expect configuration options
